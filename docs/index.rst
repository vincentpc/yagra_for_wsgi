Yagra -- Yet Another Gravatar
=====

What
===============


一个提供avatar hosting服务的站点

你可以在yagar上管理你的头像

功能:

* 注册（用户名，密码）

* 登录/登出

* 密码修改

* 上传/更换 头像

* 提供头像访问API
